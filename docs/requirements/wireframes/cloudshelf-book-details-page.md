# Book Details Page Wireframe

**Purpose**: Show detailed information about a selected book.

### Text Description

- **Header**:
  - Logo (top-left)
  - Search bar (center)
  - Shopping cart icon (top-right)
- **Main Content**:
  - Left: Large book cover image
  - Right:
    - Title
    - Author
    - Price
    - Description
    - "Add to Cart" button

```
+------------------------------------------------+

+------------------------------------------------+
```
