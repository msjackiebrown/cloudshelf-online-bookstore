# Book Catalog Page Wireframe

**Purpose**: Display a list of books for browsing.

### Text Description

- **Header**:
  - Logo (top-left)
  - Search bar (center)
  - Shopping cart icon (top-right)
- **Main Content**:
  - Grid layout of book cards:
    - Book cover image
    - Title
    - Author
    - Price
    - "View Details" button
